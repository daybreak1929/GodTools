# GodTools

## 约定

-  ActorAsset.showIconInspectWindow_id 
使用 Dictionary<string, int>
通过 Newtonsoft.Json.JsonConvert 序列化后的字符串表示生产该生物的资源消耗. 

在0.22.9版本中该字段为自由字段.
- BuildingAsset.sprite_path 使用 Dictionary<string, int>
  通过 Newtonsoft.Json.JsonConvert 序列化后的字符串表示生产该生物的资源消耗.
  
在0.22.9版本中该字段在loadSprites之后为自由字段.
